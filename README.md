# Stay Connected
Stay connected using socket.io to create live chat room where it can connecting 2 or more peoples together

Tools that used to make this app:
- Socket.io
- ReactJS
- NodeJS with Express

## How To Connect
Open this link to start using stay connected app

https://614aec221c4f677f86a0d1a3--reverent-morse-38751f.netlify.app/

Insert name and room, share your room name to your friends to connect


## Installation
First install dependencies using
```bash
npm i
```

And then change directory to each server and client directory and run npm start
```bash
cd server
npm start
cd client
npm start
```
